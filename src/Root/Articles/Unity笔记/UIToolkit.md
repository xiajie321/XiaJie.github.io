# Unity UIToolkit 笔记

## 前言

UIToolkit在最新版Unity6中已经把UIToolkit本身存在的缺陷解决的差不多了完全可以实际的应用在游戏UI上,如果是中国大陆的用户只建议用于Unity的编辑器拓展或者游戏屏幕UI的UGUI替代(<u>如果有必须使用Shader的情形并且在Unity6以下版本,建议还是仅作为制作Unity编辑器拓展工具使用</u>)。

用于Unity编辑器拓展UIToolKit相较于Odin来说,可能便捷性没有那么高,但最大的好处在于：免费、序列化可视化可以自定义样式,自由度很高。

如果想要做出效果得需要了解USS样式(类似Web中的CSS)，但相较于Shader来说上手难度没有那么高(虽然也支持使用Shader做效果,<mark>注意:Unity6及以下版本不支持使用Shader</mark>)。
